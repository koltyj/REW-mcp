Input schema (Zod):
```typescript
export const ApiCalibrateSPLInputSchema = z.object({
  action: z.enum(['start', 'check', 'stop'])
    .describe('Calibration action: start (begin pink noise), check (read SPL and get guidance), stop (end calibration)'),

  target_spl: z.number().min(70).max(95).default(85).optional()
    .describe('Target SPL in dB (default: 85 dB - broadcast reference)'),

  tolerance_db: z.number().min(0.5).max(5).default(1.0).optional()
    .describe('Acceptable tolerance in dB (default: +/-1 dB - Class 2 professional)'),

  weighting: z.enum(['A', 'C', 'Z']).default('C').optional()
    .describe('SPL meter weighting (C=flat low freq, recommended for calibration)'),

  meter_id: z.number().int().min(1).max(4).default(1).optional()
    .describe('SPL meter ID (1-4)')
});
```

Result interface:
```typescript
export interface ApiCalibrateSPLResult {
  action: string;
  success: boolean;
  message: string;
  current_spl?: number;
  target_spl?: number;
  adjustment_needed?: number;
  within_tolerance?: boolean;
  should_block_measurement?: boolean;  // true if not within tolerance
  guidance?: string;
  weighting?: string;
}
```

Execute function with three actions:

**start action:**
1. Configure generator: signal = "Pink noise" (discover exact name via getGeneratorSignals() if needed, fallback to "Pink noise"), level = -20 dBFS
2. Start generator: executeGeneratorCommand('Play')
3. Wait 2 seconds for stabilization (use setTimeout Promise)
4. Configure SPL meter: weighting from input, filter = 'Slow'
5. Start SPL meter: executeSPLMeterCommand(meterId, 'Start')
6. Wait 1 second for meter averaging
7. Return success with message "Pink noise playing. SPL meter active."

**check action:**
1. Read SPL meter: getSPLMeterLevels(meterId)
2. Return error if levels null
3. Calculate adjustment = target_spl - current_spl
4. Determine within_tolerance = Math.abs(adjustment) <= tolerance_db
5. Generate guidance:
   - If within tolerance: "SPL calibrated: {current} dB{weighting} (within +/-{tolerance} dB tolerance)"
   - If adjustment > 0: "Increase monitor volume by ~{abs(adjustment)} dB, then check again"
   - If adjustment < 0: "Decrease monitor volume by ~{abs(adjustment)} dB, then check again"
6. Return result with all fields

**stop action:**
1. Stop generator: executeGeneratorCommand('Stop')
2. Stop SPL meter: executeSPLMeterCommand(meterId, 'Stop')
3. Return success with message "Calibration stopped. Generator and SPL meter stopped."

Error handling: Follow same pattern as api-generator.ts (ZodError, REWApiError, generic Error).

Use try/finally in start action to stop generator if SPL meter fails.
