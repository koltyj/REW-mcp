<task type="auto">
  <name>Task 1: Create session state module</name>
  <files>src/session/session-state.ts</files>
  <action>
Create session state module with:

1. SessionMeasurement interface:
   - uuid?: string (REW measurement UUID, assigned after measurement)
   - name: string (auto-generated name)
   - channel: 'left' | 'right' | 'sub'
   - timestamp: number

2. SessionState interface:
   - session_id: string (UUID)
   - created_at: number (timestamp)
   - sequence_step: 'idle' | 'measuring_left' | 'measuring_right' | 'measuring_sub' | 'complete'
   - measurements: SessionMeasurement[]
   - target_spl?: number (optional, from calibration)
   - notes?: string (user description)

3. Module-level Map storage:
   - const activeSessions = new Map<string, SessionState>()

4. Functions:
   - createSession(notes?: string): SessionState
     - Generate ID with crypto.randomUUID()
     - Initialize with sequence_step: 'idle', empty measurements
     - Store in Map and return

   - getSession(sessionId: string): SessionState
     - Throw Error if not found with message including suggestion to use get_status

   - updateSession(sessionId: string, updates: Partial<SessionState>): SessionState
     - Clone session to avoid mutation
     - Merge updates
     - Store updated session

   - listActiveSessions(): SessionState[]
     - Return Array.from(activeSessions.values())
     - Sort by created_at descending (most recent first)

   - endSession(sessionId: string): void
     - Delete from Map (no-op if not found)

Use native crypto.randomUUID() (Node 14.17+, no npm dependency).
  </action>
  <verify>TypeScript compiles without errors: `npx tsc --noEmit src/session/session-state.ts`</verify>
  <done>Session state module exports SessionState interface and all 5 management functions</done>
</task>
